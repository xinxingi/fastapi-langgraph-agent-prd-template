# 安全策略

## 支持的版本

使用本节告知用户您的项目中哪些版本当前正在接受安全更新支持。

| 版本    | 支持状态           |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## 报告漏洞

使用本节告知用户如何报告漏洞。

告诉他们应该去哪里报告，他们可以多久获得一次已报告漏洞的更新，
如果漏洞被接受或拒绝会发生什么等。
